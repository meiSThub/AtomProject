## 让高频使用的命令更简单
功能说明：设置指令的别名。
语　　法：alias [别名]=[指令名称]
参　　数 : 若不加任何参数，则列出目前所有的别名设置。

## 使用方法
1. 新建或打开 ~/.bashrc
``` java
vim ~/.bashrc
```
输入以下内容，这是git常用的几个命令。

``` java
alias pull="git pull"
alias commit="git commit"
alias push="git push"
alias branch="git branch"
alias check="git checkout"
alias st="git status"

// 执行demo的时候，就自动切换到工程目录下了
alias demo="cd 项目地址"
```

2. 让别名生效
``` java
source ~/.bashrc
```

3、让别名永久生效，新建或打开 ~/.bash_profile。 （~/.bash_profile 是terminal每次打开时就会运行的？）

``` java
vim ~/.bash_profile
```
在新的一行写入

``` java
source ~/.bashrc
```
如此就可以愉快的使用简短的别名操作啦~
如果不设进行第三步骤，重启terminal，我们设置好的别名就失效了，需要手动source ~/.bashrc
